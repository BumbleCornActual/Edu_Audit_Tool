# Education Audit Tool — Flash Drive

## How to get started

### On a Windows computer:
1. Open this flash drive in File Explorer
2. Double-click **`START_WINDOWS.bat`**
3. Follow the on-screen steps
4. The tool will open automatically in your browser

### On a Mac:
1. Open this flash drive in Finder
2. Double-click **`START_MAC.command`**
   - If Mac asks "Are you sure?", click **Open**
3. Follow the on-screen steps
4. The tool will open automatically in your browser

---

## First time only
The first time you run the wizard it will:
- Check that Python is installed (and tell you how to get it if not)
- Install the required software packages (~2–5 minutes)
- Ask for your Anthropic API key (needed for summaries)
- Open the tool in your browser

After the first setup, it launches in about 10 seconds.

---

## Using the tool
1. Click **Process Files** and drop in your audio or document file
2. Wait for processing to finish (you'll see a progress bar)
3. Click **Download PDF Report** when it appears
4. Reports, transcripts, and summaries are saved in the `output/` folder

---

## Important
- Keep the terminal/command window open while using the tool
- Close that window when you're done to stop the server
- Your API key is saved in `config.txt` — don't share that file
