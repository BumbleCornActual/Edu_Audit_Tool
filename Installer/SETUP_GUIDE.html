Install Python (first time only)
Python is the programming language the tool runs on. If it's already installed, the setup script will detect it automatically.

If not, download it here:
Download Python for Windows

The tool uses AI to summarize transcripts. You need a free API key from Anthropic.

Run the setup script
Open the flash drive in File Explorer and double-click:

The script will:

  • Install all required packages automatically
  • Ask you to paste your API key

That's it!

The tool will open at http://localhost:5000 in your browser.

Common Questions

The tool opened but summarization failed — what's wrong?
Your API key is either missing or incorrect. Open config.env on the flash drive in Notepad (Windows) or TextEdit (Mac) and make sure the line reads ANTHROPIC_API_KEY=sk-ant-... with your actual key after the equals sign.
The browser says "This site can't be reached" at localhost:5000
The server isn't running. Make sure the Terminal or Command Prompt window from the start script is still open. If you closed it, double-click the start script again to relaunch.
Where do the output files go?
All output is saved to the output/ folder on the flash drive: transcripts in output/transcripts/, summaries in output/summaries/, and PDF reports in output/reports/. Processed source files are archived in output/organized/ by date.
Can I use the tool without internet?
Partially. Transcription (audio and document extraction) works offline. AI summarization requires an internet connection to reach the Anthropic API. Compliance scanning is fully offline.
Scanned images aren't being transcribed
Image files (PNG, JPG, TIFF) require Tesseract OCR to be installed separately. On Windows, download it from github.com/UB-Mannheim/tesseract. On Mac, run brew install tesseract in Terminal.
How do I add my own compliance keywords?
Once the tool is running, click Settings in the top navigation bar. You can add and remove keywords there — changes save automatically.
